Symmetry-partition-sort

This is the fastest sorting known so far

Download this project as a .zip file Download this project as a tar.gz file

Date : 2011/03/04, version 2.0 Copyright (C) 2012, Jingchao Chen

This is a C++ source code for Symmetry Partition Sort, which is the fastest practical sorting known so far. See paper http://onlinelibrary.wiley.com/doi/10.1002/spe.851/abstract

For the other version of this library, see http://www.gtoal.com/languages/bcpl/BCPL/bcplprogs/sort/ or https://docs.google.com/file/d/0BxAfEASHYgsIUFE0amdRWTFFbXM/edit?pli=1
